Welcome to the OctoCat-Splat Plug-In

WHAT DOES THIS PLUG-IN DO

'OctoCat-Splat' is a jQuery plugin reload button. But not just any reload button - it's funny-looking Octocat animating across your browser window, while simultaneously "inking" the page until the screen is black! Oh, and following a page-reload, of course. The image of Octocat is a PNG and the ink-blots is a SVG. The user has an option to change the color of the SVG color fill in the CSS file, as well as change the two images if they wish to.

HOW TO IMPLEMENT IT

Download the following files

  • octoSplat.js
  • octoSplat.css

Once you have included these files into your own working folders, be sure to include a script tag in your HTML file, right before your closing body tag with the appropriate path to the folder you have stored the plug-in file. See example below. 

<script src="octoSplat.js"></script>

Set an id to your reload button tag in your HTML file like so and you can style it however you like!

<button id="reloadButton">Reload</button>

OPTIONS

SVG Fill Color

You can also change the fill on the SVG ink-blots. The default fill colour is set to black but it can be overwritten in the SVG path, giving a the fill property a different color value. the path can be found in the octoSplat.js file.

var $ink = $('<svg fill="black" xmlns="http://www.w3.org/2000/svg"...>)
